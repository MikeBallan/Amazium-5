Typography

A consistent hierarchy is essential for good design. Amazium provides a full range of heading styles (h1–h6) alongside matching utility classes (.h1–.h6) so you can apply visual styles independently of semantic HTML.

H1 heading: 40px/48px

H2 heading: 36px/44px

H3 heading: 32px/40px

H4 heading: 28px/36px

H5 heading: 24px/32px
H6 heading: 20px/28px

Body Copy: 16px/24px

Small Label: 12px/16px

How It Works

Typography in Amazium is powered by @font-face and design tokens.

@font-face {
  font-family:'txt-body';
  src:url('../font/your-font-regular.woff') format('woff');
  font-weight:400;
  font-style:normal;
  font-display:swap;
}

@font-face {
  font-family:'txt-title';
  src:url('../font/your-font-heading.woff') format('woff');
  font-weight:400;
  font-style:normal;
  font-display:swap;
}

Utilities

Quick helper classes for common styling.

Utility Class: Description: Example:
.show--mobile Displays an element on mobile devices only.
.show--tablet Displays an element on tablets only.
.show--screen Displays an element on screens (desktop).
.hide--mobile Hides an element on mobile devices.
.hide--tablet Hides an element on tablets.
.hide--screen Hides an element on desktop screens.
