# Amazium
A lightweight, modern CSS framework built with **CSS Grid + Flexbox**, designed for fast prototyping and clean, responsive UI development.

Built and maintained by Mike Ballan.

---

## What is Amazium?
Amazium is a minimalist CSS framework focused on **speed, structure, and flexibility**.

It provides a solid foundation for building responsive interfaces without unnecessary complexity or framework lock-in.

No build tools required. No JavaScript dependency. Just CSS.

---

## Key Features

### Grid System
A fully responsive 12-column grid system with support for:

- Fixed and fluid layouts
- Column spanning
- Offsets
- Nested grids
- Responsive overrides (mobile / tablet / desktop)
- Column ordering utilities

---

### Design Foundations
A consistent design system built on CSS variables:

- Typography scale
- Colour system (design tokens)
- Spacing system
- Radius system
- Layout structure

---

### Components
Reusable UI components including:

- Buttons
- Forms (inputs, selects, checkboxes, radios, switches, toggles)
- Messages / alerts
- Tables (dynamic / structured layouts)

---

### Utilities
Helper classes for rapid UI building:

- Text alignment
- Display utilities
- Visibility controls
- Reset utilities
- Responsive show / hide rules
